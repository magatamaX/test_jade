_top.jade

jade（pug）の記法

jade（pug）の記法はSASS等と似ており、ネストによって記述します。

基本的なHTMLの記法とだいぶ異なるので最初はとっつきにくいですが、慣れてくると直感的でコードも短く済み、便利です。

PHP等の動的なコードを埋め込む際は使いにくいかも。静的HTMLの大量生産に向いてるかな～^^

つかいかた

gulp-jadeをインストール。で、

gulp.task('jade', function(){
return gulp.src(['./src/jade/**/*.jade', '!./src/jade/**/_*.jade'])
.pipe(jade({
pretty: true
}))
.pipe(gulp.dest('./'));
});

こんな感じで設定。
